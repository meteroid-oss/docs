## Matrix Pricing updates

### Recently added/removed dimensions' pricing updates

If a product utilizes a Matrix Pricing model, and if you updated a Metric associated to it (e.g adding new dimensions) in the Metrics section, you can manage the pricings associated to new dimensions from the product details page.

By clicking on a product and navigating to the Matrix Dimensions section, you can monitor the status of your dimension mappings:
* Active: A price is correctly associated with this dimension.
* Missing: No pricing has been defined for this dimension. This typically occurs after adding a metric's dimensions in the Metrics section.To resolve this, click "Add Missing", input the required pricing, and select "Preview" then "Apply". This action will automatically update the pricing across all plans currently using this product.
* Orphaned: This status appears if a dimension has been deleted from the original metric but still has legacy pricing associated with it in the product. You can remove them by clicking "Remove orphaned"

### Already existing dimensions' pricing updates

To update already existing dimensions' pricing, a new plan version should be created (cf. below).

## Updating Non-Matrix Pricing

For products that do not use a Matrix pricing model (cf. above for this case), price updates are managed through Plan Versioning. This ensures that price changes are controlled and do not unintentionally disrupt active billing cycles.

To update pricing for a standard plan:
* Navigate to Plans and select the plan you wish to modify.
* Click on New Version.
* Add, remove, or update specific Pricing Components.
* Once your changes are complete, click Publish Version.

By default, Meteroid handles versioning as follows:
* Existing Subscriptions: Subscriptions already active will stick to the plan version they were created with. This protects existing customers from sudden price changes.
* New Subscriptions: Any new subscription created after the publication will automatically use the latest plan version.

If you wish to move an existing customer to the latest Plan version: 
* Go to the Subscriptions section and click on the specific subscription.
* Click Actions > Change Plan.
* Select the same Plan name. By default, Meteroid will suggest the latest Plan version.
      
💡**Note:** For more informations on Plan change, please read the [updating a subscription](/subscriptions/updating-a-subscription#changing-the-plan-associated-to-a-subscription) section.
